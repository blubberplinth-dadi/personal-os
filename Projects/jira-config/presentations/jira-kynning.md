# Glærudrög: JIRA Kynning fyrir STM

## Glæra 1: Titill
- JIRA hjá Stafrænu TM
- Kynning fyrir Bót og Ronju
- Febrúar 2026

## Glæra 2: Tvenns konar borð
- **Epic borð** → Forgangsröðun, greining, niðurbrot verkefna…
- **Þróunarborð** → Þróun, dagleg vinna…
- Dæmi: Standup = þróunarborð, Skipulagsfundur = Epic borð
- [Sýna bæði borð í JIRA]

## Glæra 3: Miðategundir
| Tegund | Notkun |
|--------|--------|
| Epic | Stór verkefni, 1-3 mánuðir |
| Story | Virkni sem skilar gildi, 1-5 dagar |
| Task | Tæknileg vinna |
| Bug | Villur |

## Glæra 4: Hvenær Epic?
- **Já:** 3+ sögur, meira en einn sprettur, spannar mörg kerfi
- **Nei:** Einföld villa, 1-5 dagar, einangrað verkefni
- Meginregla: Saga >5 dagar = sennilega of stór

## Glæra 5: Vörur / Kerfi (Components)
- Veldu alltaf vöru: Vefsala TM, Bóthildur, Rósin, API, o.fl.
- Mörg kerfi? Veldu öll sem verkefnið snertir

## Glæra 6: Búa til nýjan miða
1. Smelltu á "Create" í JIRA
2. Veldu tegund (Epic / Story / Task / Bug)
3. Fylltu út Summary, Teymi, Component
4. Bættu við Epic Link ef miðinn tilheyrir Epic
5. Vistaðu miðann
- **Ath:** Miði byrjar í "To Do" → birtist á Epic borði
- Ef miðinn á að fara beint á þróunarborð: breyttu Status í "Next In" eða "In Progress"

## Glæra 7: Hvar birtist hvað?
| | Epic borð | Þróunarborð |
|--|-----------|-------------|
| Epic | Já | Nei |
| Story tengd Epic (í þróun) | Nei | Já |
| Sjálfstætt (skipulag) | Já | Nei |
| Sjálfstætt (í þróun) | Nei | Já |

## Glæra 8: Gátlisti fyrir nýjan miða
- Tegund (Epic / Story / Task / Bug)
- Teymi (Bót eða Ronja)
- Vara / kerfi (Component)
- Epic Link (ef við á)
- Status (ef beint á þróunarborð)

## Glæra 9: Sýnikennsla
- Búa til Epic og tengdar sögur
- Færa miða á milli
- Quick Filters

## Glæra 10: Næstu skref
1. Í dag afritar fólk sína miða yfir í JIRA. Daði mun færa aðra miða.
2. Á morgun verður JIRA notað í standup.
3. Sýnum þolinmæði og lærum saman hvað mætti virka betur :-)
